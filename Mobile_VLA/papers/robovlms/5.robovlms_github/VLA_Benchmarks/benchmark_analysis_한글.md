# RoboVLMs VLA 벤치마크 분석 (한글)

## 벤치마크 개요

RoboVLMs는 시뮬레이션 환경과 실제 로봇 조작 작업을 포함한 여러 벤치마크에 대한 포괄적인 평가를 제공합니다. 평가 과정은 엄격하고 재현 가능하며 포괄적으로 설계되었습니다.

## CALVIN 벤치마크

### 데이터셋 특성
- **총 시연**: 24K 인간 원격 조작 시연
- **기본 기술**: 34개 사전 정의된 조작 작업
- **궤적 길이**: 궤적당 64 시간 단계 미만
- **언어 지시**: 자연어 설명으로 주석 처리

### 작업 카테고리
1. **블록 조작**: 블록 회전, 이동, 들어올리기, 배치
2. **슬라이더 작업**: 슬라이더 좌우 이동
3. **조명 제어**: 전구 및 LED 켜기/끄기
4. **서랍 작업**: 서랍 열기/닫기, 서랍 밀어넣기
5. **쌓기 작업**: 쌓인 블록 들어올리기

### 평가 분할
- **장면 A, B, C, D**: 다른 장면 구성
- **훈련 분할**: ABCD (전체 훈련), ABC (일반화)
- **테스트 분할**: D (보이지 않는 장면 평가)

## SimplerEnv 벤치마크

### 환경 설정
- **목적**: 실물-시뮬레이션 환경 평가
- **로봇 플랫폼**: Google Robot, WidowX+Bridge
- **작업 유형**: Pick, move, open/close, place 작업
- **평가**: 교차 구현 일반화 테스트

### Google Robot 작업
1. **Pick Coke Can**: 테이블에서 빈 콜라 캔 들어올리기
   - **위치**: 수평, 수직, 서 있는 자세
   - **그리드 포인트**: 방향당 25개 위치
   - **총 시행**: 75개 실험

2. **Move {obj1} near {obj2}**: 객체 조작
   - **객체**: 8개 다른 객체 (병, 캔, 과일)
   - **구성**: 삼각형 배치
   - **총 시행**: 60개 실험

3. **Open/Close Drawer**: 관절 객체 조작
   - **서랍**: 상단, 중간, 하단
   - **작업**: 열기/닫기 작업
   - **총 시행**: 54개 실험

4. **Open Top Drawer & Place Apple**: 순차 조작
   - **단계**: 서랍 열기, 사과 배치
   - **위치**: 3개 로봇 위치, 9개 사과 위치
   - **총 시행**: 27개 실험

### WidowX+Bridge 작업
1. **Put Spoon on Towel**: 객체 배치
   - **설정**: 15cm 정사각형 구성
   - **방향**: 수평 및 수직
   - **총 시행**: 24개 실험

2. **Put Carrot on Plate**: 숟가락 작업과 유사
   - **객체**: 당근과 접시
   - **구성**: 숟가락 작업과 동일
   - **총 시행**: 24개 실험

3. **Stack Green Block on Yellow Block**: 블록 쌓기
   - **블록 크기**: 3cm 블록
   - **정사각형 크기**: 10cm 및 20cm
   - **총 시행**: 24개 실험

4. **Put Eggplant in Yellow Basket**: 컨테이너 조작
   - **설정**: 싱크대 분지 구성
   - **객체**: 가지와 노란 바구니
   - **총 시행**: 24개 실험

### 성능 결과

#### WidowX+Bridge 성능
- **Put Spoon on Towel**: 70.8% 성공률
- **Put Carrot on Plate**: 45.8% 성공률
- **Stack Green Block on Yellow Block**: 33.3% 성공률
- **Put Eggplant in Yellow Basket**: 20.8% 성공률

#### Google Robot 성능
- **Pick Coke Can**: 94.0% 성공률
- **Move Near**: 90.3% 성공률
- **Open/Close Drawer**: 77.3% 성공률
- **Open Drawer & Place Apple**: 43.5% 성공률

### 주요 성과
- **교차 구현 일반화**: 다양한 로봇 플랫폼에서 강력한 성능
- **다중 작업 능력**: 다양한 조작 작업의 효과적인 처리
- **실물-시뮬레이션 전이**: 시뮬레이션에서 현실로의 성공적인 전이

## 실제 로봇 실험

### 실험 설정
- **로봇 플랫폼**: Kinova Gen-3 7-DoF 팔과 Robotiq 2F-85 그리퍼
- **카메라**: 정적 작업공간 카메라 + 손목 장착 카메라
- **작업공간**: 55cm × 24cm 테이블, 40개 이상의 객체
- **작업**: 20개 조작 작업, 각각 5회 롤아웃

### 평가 설정
1. **Simple Setting**: 훈련 분포와 유사한 장면
2. **Unseen Distractor**: 이전에 보지 못한 방해 객체 도입
3. **Unseen Background**: 훈련 데이터에 없던 새로운 테이블보 추가
4. **Unseen Objects**: 훈련 데이터셋에 포함되지 않은 객체 조작
5. **Novel Skill Description**: GPT-4로 생성된 지시 동사 동의어

### 작업 카테고리
- **열기/닫기**: 서랍, 오븐, 토스터 작업
- **Pick & Place**: 객체 조작 및 배치
- **버튼 누르기**: 스위치 및 버튼 작업
- **순차 작업**: 다단계 조작 시퀀스

### 성능 결과

#### 전체 성능
- **Simple Setting**: 75% 성공률
- **Unseen Distractor**: 60% 성공률
- **Unseen Background**: 50% 성공률
- **Novel Skill Description**: 55% 성공률
- **Unseen Objects**: 33% 성공률

#### 작업별 성능
- **Open Drawer**: 75% 성공률
- **Pickup Eggplant**: 60% 성공률
- **Press Toaster**: 50% 성공률
- **Pickup Knife**: 45% 성공률
- **Pickup Cucumber**: 40% 성공률

### 주요 성과
- **실제 배포**: 성공적인 실제 로봇 조작
- **일반화**: 보이지 않는 설정에서 강력한 성능
- **자기 수정**: 궤적 수정 기능의 출현
- **견고성**: 다양한 시나리오에서 효과적인 처리

## 주요 통찰

### 1. VLA 효과성
- **우수한 성능**: VLA 모델이 비VLA 베이스라인을 능가
- **일반화**: 보이지 않는 시나리오에서 강력한 제로샷 성능
- **장기**: 연속 작업의 효과적인 처리

### 2. 아키텍처 영향
- **정책 헤드**: 인터리브 모델링보다 더 효과적
- **히스토리 통합**: 성능을 위한 다단계 관찰 처리 중요
- **액션 공간**: 장기 작업에 대해 연속 액션이 이산보다 우수

### 3. 훈련 전략
- **사전 훈련**: 교차 구현 데이터가 일반화 개선
- **파인튜닝**: 작업별 성능을 위한 도메인 내 데이터 필수
- **데이터 효율성**: VLA 모델이 강력한 퓨샷 학습 능력 보여줌

### 4. 실제 적용 가능성
- **배포 성공**: 성공적인 실제 로봇 조작
- **일반화**: 보이지 않는 객체 및 시나리오의 효과적인 처리
- **자기 수정**: 궤적 수정을 위한 출현 능력
- **견고성**: 다양한 평가 설정에서 강력한 성능
