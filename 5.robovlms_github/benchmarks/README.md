# RoboVLMs Benchmarks Analysis

## 벤치마크 개요

### 1. CALVIN 벤치마크

#### 데이터셋 특성
- **총 시연**: 24K 인간 텔레오퍼레이션 시연
- **언어 지시**: 모든 시연에 언어 지시 포함
- **궤적 길이**: 64 시간 단계 이하
- **기본 기술**: 34개 사전 정의된 기본 기술

#### 34개 기본 기술 분류

##### 블록 조작 (Block Manipulation)
1. rotate blue block right
2. rotate blue block left
3. rotate red block right
4. rotate red block left
5. lift red block slider
6. lift blue block slider
7. lift pink block slider
8. lift blue block drawer
9. lift blue block table
10. lift pink block table
11. lift stacked blocks
12. push blue block left
13. push blue block right
14. push red block left
15. push pink block left
16. push pink block right

##### 슬라이더 조작 (Slider Manipulation)
17. move slider right
18. move slider left
19. place slider

##### 서랍 조작 (Drawer Manipulation)
20. open drawer
21. close drawer
22. push in drawer
23. place in drawer

##### 조명 조작 (Light Manipulation)
24. turn on light bulb
25. turn off light bulb
26. turn on led light
27. turn off led light

#### 분할 설정
- **A, B, C, D**: 장면 설정별 분할
- **훈련 분할**: ABCD, ABC
- **테스트 분할**: D (unseen scene)
- **평가**: 1000 롤아웃, 5개 연속 하위 작업

#### 성능 메트릭
- **연속 작업 성공률**: 1~5개 연속 작업 완료 성공률
- **평균 달성 작업 수**: 각 롤아웃에서 평균적으로 달성한 작업 수

### 2. SimplerEnv 벤치마크

#### WidowX+Bridge 환경

##### Put Spoon on Towel
- **설정**: 15cm x 15cm 정사각형 테이블
- **스푼 위치**: 정사각형 한 모서리
- **타월 위치**: 다른 모서리
- **방향**: 수평/수직 교대
- **총 시행**: 24회

##### Put Carrot on Plate
- **설정**: Put Spoon on Towel과 유사
- **스푼 → 당근**: 스푼을 당근으로 교체
- **타월 → 접시**: 타월을 접시로 교체

##### Stack Green Block on Yellow Block
- **설정**: 10cm, 20cm 정사각형 구성
- **녹색 블록**: 한 모서리
- **노란색 블록**: 다른 모서리
- **블록 크기**: 3cm
- **총 시행**: 24회

##### Put Eggplant in Yellow Basket
- **설정**: 싱크의 좌우 대야
- **가지**: 우측 대야에 무작위 위치
- **노란 바구니**: 좌측 대야
- **총 시행**: 24회

#### Google Robot 환경

##### Pick Coke Can
- **환경**: 방해 요소 없는 표준 구성
- **캔 위치**: 25개 그리드 포인트
- **방향**: 수평(누워), 수직(누워), 서 있는 상태
- **총 시행**: 75회 (25 x 3 방향)

##### Move {obj1} near {obj2}
- **설정**: 3개 객체를 삼각형으로 배치
- **역할**: 소스, 타겟, 방해 요소
- **객체**: 8개 중 5개 삼중항 무작위 선택
- **패턴**: 정삼각형, 역삼각형
- **총 시행**: 60회

##### Open/Close Drawer
- **설치**: 3개 서랍이 있는 캐비닛
- **로봇 위치**: 9개 그리드 위치
- **동작**: 열기/닫기
- **총 시행**: 54회

##### Open Top Drawer & Place Apple
- **작업**: 서랍 열기 + 사과 이동
- **로봇 위치**: 3개 위치
- **사과 위치**: 캐비닛 표면 9개 그리드 포인트
- **총 시행**: 27회

### 3. 실제 로봇 벤치마크

#### 하드웨어 구성
- **로봇 팔**: Kinova Gen-3 (7-DoF)
- **그리퍼**: Robotiq 2F-85 병렬 턱 그리퍼
- **카메라**: 
  - Kinect Azure (정적 카메라)
  - RealSense D435i (손목 카메라)
- **작업 공간**: 55cm x 24cm 테이블
- **객체**: 40개 이상의 다양한 객체

#### 평가 설정

##### Simple 설정
- **목적**: 훈련 데이터 분포에 대한 모델 적합성 평가
- **특징**: 훈련 데이터와 유사한 장면
- **예상 성능**: 높은 성공률

##### Unseen Distractor 설정
- **목적**: 이전에 보지 못한 방해 객체 도입
- **특징**: 조작 객체는 훈련 데이터에 포함
- **도전**: 새로운 방해 요소에 대한 적응

##### Unseen Background 설정
- **목적**: 배경 변경
- **방법**: 훈련 데이터에 없는 새로운 테이블보 추가
- **특징**: 색상과 패턴이 다른 테이블보
- **도전**: 시각적 맥락 변화에 대한 적응

##### Unseen Object 설정
- **목적**: 훈련 데이터에 없는 객체 조작
- **특징**: Unseen Distractor와 동일한 객체 사용
- **도전**: 새로운 객체에 대한 일반화

##### Novel Skill Description 설정
- **목적**: 새로운 동사로 작업 지시 변경
- **방법**: GPT-4로 동사의 동의어 3개 생성
- **예시**: 
  - "press" → "hit"
  - "pick up" → "take"
  - "close" → "shut"
  - "pour" → "sprinkle"
- **도전**: 언어적 일반화

## 벤치마크별 특성 분석

### 1. CALVIN의 특징
- **장점**: 체계적인 분할, 다양한 작업, 표준화된 평가
- **한계**: 시뮬레이션 환경, 제한된 작업 복잡도
- **적용**: VLA 구조 비교, 일반화 능력 평가

### 2. SimplerEnv의 특징
- **장점**: 실제-시뮬 환경, 다양한 로봇 설정
- **한계**: 제한된 작업 수, 특정 환경에 특화
- **적용**: Cross-embodiment 데이터 효과 평가

### 3. 실제 로봇의 특징
- **장점**: 실제 환경, 다양한 설정, 실용적 평가
- **한계**: 높은 비용, 제한된 반복 수
- **적용**: 실제 배포 가능성 평가

## 평가 메트릭

### 1. 성공률 (Success Rate)
- **정의**: 작업 완료 비율
- **계산**: 성공한 롤아웃 수 / 전체 롤아웃 수
- **범위**: 0.0 ~ 1.0

### 2. 평균 달성 작업 수 (Average Length)
- **정의**: 각 롤아웃에서 평균적으로 달성한 작업 수
- **계산**: Σ(달성한 작업 수) / 전체 롤아웃 수
- **범위**: 0.0 ~ 5.0 (CALVIN 기준)

### 3. 연속 작업 성공률
- **정의**: 1~5개 연속 작업 완료 성공률
- **의미**: 장기간 일관성 평가
- **중요성**: 실제 로봇 작업에서 중요

## 벤치마크 선택 기준

### 1. CALVIN 선택 이유
- **표준화**: VLA 연구에서 널리 사용
- **체계성**: 명확한 분할과 평가 기준
- **다양성**: 34개 기본 기술 포함

### 2. SimplerEnv 선택 이유
- **실제성**: 실제-시뮬 환경
- **확장성**: 다양한 로봇 설정 지원
- **비교성**: 기존 연구와의 공정한 비교

### 3. 실제 로봇 선택 이유
- **실용성**: 실제 배포 가능성 평가
- **일반화**: 다양한 설정에서 평가
- **신뢰성**: 실제 환경에서의 성능 검증

## 향후 벤치마크 발전 방향

### 1. 더 복잡한 작업
- **장기간 작업**: 더 긴 시퀀스의 작업
- **다단계 작업**: 여러 단계를 거치는 복잡한 작업
- **동적 환경**: 변화하는 환경에서의 작업

### 2. 더 다양한 환경
- **가정 환경**: 실제 가정 환경 시뮬레이션
- **산업 환경**: 제조업 환경
- **야외 환경**: 야외 작업 환경

### 3. 더 정교한 평가
- **인간 수준 평가**: 인간과의 성능 비교
- **효율성 평가**: 시간, 에너지 효율성
- **안전성 평가**: 안전한 작업 수행 능력
