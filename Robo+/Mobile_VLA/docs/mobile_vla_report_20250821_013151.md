## 📊 성능 지표 공식

### MAE (Mean Absolute Error)
```
MAE = (1/n) * Σ|y_true - y_pred|
```

### RMSE (Root Mean Squared Error)
```
RMSE = √((1/n) * Σ(y_true - y_pred)²)
```

### R² Score
```
R² = 1 - (SS_res / SS_tot)
```

### Threshold Accuracy
```
Accuracy_t = (1/n) * Σ(|y_true - y_pred| ≤ t)
```
