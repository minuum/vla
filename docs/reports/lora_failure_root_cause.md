## 🔍 문제 요약

### 증상

| 모델 | Left Acc | Right Acc | Avg |
|:---|:---:|:---:|:---:|
| **Frozen L+R** | 85% | 100% | **92.5%** |
| **LoRA L+R** | 100% | 0% | 50% |

### 원인

**LoRA 모델은 **Left만 있는 데이터**로 학습됨!**

## 🎯 결론

### LoRA 실패 원인

> **LoRA 자체의 기술적 문제가 아님!**
> 
> 학습 데이터의 **99%가 Left였기 때문에**,  
> 모델이 "모든 입력에 대해 Left 방향(양수) 출력"을 학습한 것.

### 증거

1. **LoRA 학습 로그**: `num_episodes: 200` (12/03 시점 데이터만)
2. **파일 분포 분석**: 12/03 파일 중 Left 250개, Right 3개
3. **모델 출력**: Left/Right 입력 모두 양수 출력 → Left만 학습한 결과

## ✅ 재실험 제안

### LoRA를 공정하게 평가하려면:

| 설정 | 값 |
|:---|:---|
| 데이터 | 500개 (L/R 균형) |
| Episode Pattern | `episode_*.h5` (전체) |
| 예상 결과 | Frozen과 유사한 성능 |

```json
}
```
