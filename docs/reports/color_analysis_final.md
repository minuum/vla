## 💡 "물빠진 색"의 원인

### 색온도 변화

**정상**: 차가운 중성 톤 (R≈G≈B)
```
Day light, 균형잡힌 화이트 밸런스
```

**오류**: 따뜻한 톤 (B << R, G)
```
그림자 또는 따뜻한 조명
파란색 빛 차단 → 노란빛/갈색 tint
```

### 인지적 "물빠진" 느낌

1. **Desaturation 아님**
   - 실제로는 채도가 높음 (측정 방법 문제)
   
2. **색온도 변화**
   - 차가운 중성 → 따뜻한 톤
   - 물빠진 옷처럼 "누렇게" 보임
   
3. **파란색 loss**
   - Blue channel: -44~48% 감소
   - 생동감 있는 색감 손실

## 🔬 기술적 분석

### RGB 채널 밸런스

**정상 프레임의 특징**:
```python
R/G ≈ 0.96-0.97  # Red가 Green보다 약간 적음
G/B ≈ 0.97-0.98  # Green이 Blue보다 약간 적음
→ 거의 중성 (neutral)
```

**오류 프레임의 특징**:
```python
R/G ≈ 0.99-1.05  # Red가 Green과 비슷하거나 많음
G/B ≈ 1.08-1.13  # Green이 Blue보다 8-13% 많음!!!
→ 파란색 부족 (Blue-deficient)
```

### 색온도 추정

**Kelvin 온도 (추정)**:
- 정상: ~5500-6000K (주간 daylight)
- 오류: ~3500-4000K (따뜻한 실내광/그림자)

**변화량**: -1500~-2500K (차가움 → 따뜻함)
