### RGB 절대값 비교

**문제 에피소드 (정상 프레임 평균)**:
```
RGB = (128.3, 135.3, 138.2)
→ Red < Green < Blue
→ 차가운 Cyan/Blue tint
```

**정상 에피소드 (정상 프레임 평균)**:
```
RGB = (128.0, 132.3, 134.8)
→ Red ≈ Green < Blue
→ 거의 중성, 약간 차가움
```

**차이**:
```
ΔR: +0.3 (Red는 거의 같음)
ΔG: +3.0 (Green이 3 증가)
ΔB: +3.5 (Blue가 3.5 증가)

→ Green/Blue만 증가 = Red 상대적 부족!
```
