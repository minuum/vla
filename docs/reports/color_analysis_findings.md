## 🔍 분석 및 해석

### 놀라운 발견

**예상**:
- ❌ 오류 프레임이 "물빠진" 색 = 낮은 채도
- ❌ 회색조에 가까운 이미지

**실제**:
- ✅ 오류 프레임이 **가장 높은 채도**
- ✅ 평균보다 38-43% 더 vivid
- ✅ 통계적으로 명확한 이상치 (평균 + 2σ 초과)

### 왜 이런 결과가?

**가설 1: 그림자의 고대비 효과**
```
정상 프레임: 균일한 조명 → 중간 채도 (22-24)
오류 프레임: 강한 그림자 + 밝은 영역 → 대비 증가 → 높은 채도 (33-34)
```

**가설 2: HSV 색공간 특성**
- 어두운 영역에서 작은 색상 차이도 높은 채도로 측정됨
- 그림자는 파란색/보라색 tint를 가질 수 있음
- 이것이 채도를 높임

**가설 3: 저채도 픽셀 비율의 모순**
```
오류 에피소드: 저채도 픽셀 74-75% (낮음)
정상 에피소드: 저채도 픽셀 79-82% (높음)

→ 오류 프레임이 오히려 "컬러풀"한 픽셀 비율이 높음
```
