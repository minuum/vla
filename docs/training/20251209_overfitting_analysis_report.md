## 과적합 판단 기준

### Gap 분석

```
Gap = Val_Loss / Train_Loss_avg

Gap 해석:
- Gap < 1: Val이 Train보다 낮음 → 이상 (데이터 유출 의심)
- Gap = 1: 동일 → 완벽한 일반화
- Gap 1~5: 정상 범위
- Gap > 5: 과적합 주의
- Gap > 10: 심각한 과적합
```

**우리 모델**: 
- 최대 Gap: 1.13 (Epoch 6)
- 모든 Epoch에서 Gap < 2
- **과적합 아님** ✅
