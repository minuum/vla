## 📊 주요 발견사항

### 1. Instruction Grounding 문제
- VLM Frozen 상태에서는 instruction을 구분하지 못함
- RoboVLMs Pretrained VLM은 이미 Robot 도메인에서 학습되어 instruction 이해 능력 있음
- Action Head만 학습해도 instruction grounding 가능성 있음

### 2. 전이학습 전략
```
HuggingFace Kosmos-2 (일반 VLM)
  ↓ [Pre-training by Microsoft]
Google Robot Data
  ↓ [Fine-tuning by RoboVLMs - VLM + Action Head 함께]
kosmos_ph_google-robot-post-train.pt
  ↓ [우리 전략: VLM Frozen, Action Head 교체]
Mobile VLA (2DoF)
```
