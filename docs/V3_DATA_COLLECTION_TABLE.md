### 🛠️ 수집 시 핵심가이드 (수동 조작)
1. **타이밍 변주**: 매 에피소드마다 꺾는 프레임, 멈추는 프레임을 의도적으로 다르게 합니다. (예: 2프레임째 꺾기, 5프레임째 꺾기)
2. **조향 강도 변주**: `L`(급격히 꺾기)과 `FL`(부드럽게 꺾기)을 의도적으로 섞어서 씁니다.
3. **오류 시퀀스(Recovery) 삽입 필수**: 바구니가 우측에 있는데 일부러 좌측으로 가다가 우측으로 다시 꺾는 식의 **회복(Recovery) 시나리오**를 10% 정도 비율로 꼭 섞어야 나중에 궤도를 벗어났을 때 로봇이 스스로 돌아올 수 있습니다.
