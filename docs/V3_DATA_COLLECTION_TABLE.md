### 🛠️ 수집 시 핵심가이드 (수동 조작)
1. **타이밍 변주**: 매 에피소드마다 꺾는 프레임, 멈추는 프레임을 의도적으로 다르게 합니다. (예: 2프레임째 꺾기, 5프레임째 꺾기)
2. **조향 강도 변주**: `L`(급격히 꺾기)과 `FL`(부드럽게 꺾기)을 의도적으로 섞어서 씁니다.
3. **오류 시퀀스(Recovery) 삽입 필수**: 모델이 현실 테스트에서 실수했을 때 다시 돌아올 수 있도록 궤도 수정 과정을 가르쳐주는 핵심 데이터입니다.
