# VLA Phase 1.5 Data Collection Plan (장애물 없음, 목표물만)

## 1. 개요 및 목적
- **목표 물체**: 회색 바구니 (Gray Basket)
- **장애물**: 없음 (Brown pot 사용 안 함)
- **핵심 목표**: 모델이 바구니의 "화면상 위치(좌/중단/우)"와 "크기(거리)"만을 시각적으로 인식하여 접근하고 멈추는(Target Reaching) 순수 능력을 학습하는 것.

## 2. 수집 시나리오 (총 300~400개 권장)

기존처럼 특정 Frame에서 고정된 액션(타이밍 암기)이 나오는 것을 막기 위해, 바구니의 **초기 위치**와 **거리**를 다양하게 조합합니다.

### A. 위치 세팅 (로봇 정면 고정)
1. **Left (좌측 편향)**: 로봇 정면에서 30~50cm 좌측
2. **Center (정중앙)**: 로봇 완벽한 정면
3. **Right (우측 편향)**: 로봇 정면에서 30~50cm 우측
4. **Extreme (극단적 좌/우)**: 화면 가장자리에만 살짝 걸쳐 있는 위치

### B. 거리(Depth) 세팅
1. **Close (1m 내외)**: 시작부터 바구니가 매우 크게 보임 → 3~5프레임 만에 도달 및 STOP
2. **Medium (2~3m)**: 표준 접근 거리 → 조향 후 8~12프레임 전진 후 STOP
3. **Far (4m 이상)**: 시작부터 바구니가 작게 보임 → 18프레임 내내 전진 또는 18프레임 초과 시나리오

## 3. 에피소드 행동 예시 (타이밍 다양화)

- **Left + Close**: `L L FL FL STOP` (빨리 끝남)
- **Left + Far**: `L FL FL F F F F F F F F F F F STOP` (길게 전진)
- **Extreme Right + Medium**: `R R FR FR FR F F F F STOP` (강한 방향 전환 후 전진)

## 4. 핵심 확인 사항 (수집자 가이드)
- **절대 안 되는 것**: 바구니 모델을 항상 2m 정면에만 두고 수집 (타이밍 암기 부활).
- **Instruction 통일**: `"Navigate directly to the gray basket"` 혹은 명칭에 맞는 단일 지시문 사용. "brown pot" 언급 제거.
