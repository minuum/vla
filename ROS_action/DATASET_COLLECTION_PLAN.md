## 📝 수집 가이드라인

### 제거된 변수 처리

1. **Variant 패턴**
   - 수집 시 Core 패턴만 사용
   - Variant는 더 이상 수집하지 않음

2. **거리 (close/far)**
   - medium 거리만 수집
   - close/far는 더 이상 수집하지 않음

3. **시간대**
   - 수집 시간은 자유롭게
   - 메타데이터로만 저장 (학습에는 사용 안 함)

### 수집 시 주의사항

- 각 시나리오당 정확히 250개 목표
- Core 패턴 가이드 준수
- medium 거리만 사용
- 18프레임 고정 (RoboVLMs 표준)
