# ⚙️ Action Pre-process 분석

> **인용**: 논문 3페이지 2번째 줄부터 5페이지 1번째 줄까지의 Methodology 섹션

## 🎯 **Action Pre-process 개요**

### **액션 전처리의 목적**
> **인용**: 논문에서 제시된 액션 전처리의 목적과 필요성

- **데이터 정규화**: 액션 데이터의 표준화
- **특징 추출**: 액션에서 의미있는 특징 추출
- **노이즈 제거**: 불필요한 노이즈 제거

## 🔧 **액션 전처리 단계**

### **1. 데이터 정규화**
> **인용**: 논문에서 설명된 데이터 정규화 방법

- **스케일링**: 액션 값의 범위 정규화
- **중심화**: 평균을 0으로 조정
- **표준화**: 분산을 1로 조정

### **2. 특징 추출**
> **인용**: 논문에서 제시된 특징 추출 방법

- **시간적 특징**: 액션의 시간적 패턴
- **공간적 특징**: 액션의 공간적 특성
- **통계적 특징**: 액션의 통계적 특성

### **3. 노이즈 제거**
> **인용**: 논문에서 설명된 노이즈 제거 방법

- **필터링**: 고주파 노이즈 제거
- **스무딩**: 액션 시퀀스 평활화
- **이상치 제거**: 비정상적인 액션 값 제거

## 📊 **액션 공간 처리**

> **인용**: 논문에서 제시된 액션 공간 처리 방법

### **Continuous Action Space**
- **특징**: 연속적인 값으로 액션 표현
- **처리**: 정규화 및 스케일링
- **장점**: 정밀한 제어 가능

### **Discrete Action Space**
- **특징**: 이산적인 값으로 액션 표현
- **처리**: 원-핫 인코딩 또는 임베딩
- **장점**: 간단한 제어

## 🔍 **액션 시퀀스 처리**

> **인용**: 논문에서 설명된 액션 시퀀스 처리 방법

### **시퀀스 정규화**
- **길이 정규화**: 시퀀스 길이 통일
- **패딩**: 짧은 시퀀스에 패딩 추가
- **트런케이션**: 긴 시퀀스 자르기

### **시간적 정렬**
- **동기화**: 관측과 액션의 시간적 동기화
- **인터폴레이션**: 누락된 시간 단계 보간
- **리샘플링**: 일정한 시간 간격으로 재샘플링
